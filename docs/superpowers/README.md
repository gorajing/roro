# Superpowers Archive

These plans and specs are historical design exploration. They are useful for
understanding how Roro got here, but they are not current execution truth.

Current truth lives in:

- [`../../HANDOFF.md`](../../HANDOFF.md) for engineering state and invariants
- [`../../PUBLIC.md`](../../PUBLIC.md) for public-readiness gates
- [`../ROADMAP.md`](../ROADMAP.md) for live sequencing
- [`../INTERACTION.md`](../INTERACTION.md) for the current interaction contract

Before using any file in this folder as implementation guidance, reconcile it
against the current code and the documents above. Several files here predate the
local-first memory2 rebuild, the job-first strategy, the default floating pet,
the release-channel guard, and the cut-from-v0 voice/cosmetics decisions.
